📍 GPS Alarm – Alarme por Geolocalização (PWA)

Aplicação Web desenvolvida como Progressive Web App (PWA) que utiliza geolocalização em tempo real para despertar o usuário antes de chegar ao seu destino.

O aplicativo é ideal para pessoas que utilizam transporte público e precisam ser alertadas antes de chegar ao local desejado, evitando perder o ponto durante o trajeto.

A aplicação funciona diretamente no navegador e também pode ser instalada no celular como um aplicativo, sem necessidade de loja de aplicativos.

O projeto foi desenvolvido utilizando HTML5, CSS3 e JavaScript puro, explorando recursos modernos da web como Geolocation API, Service Worker e Web App Manifest.

🎯 Funcionalidades

• 📍 Detecção da localização do usuário em tempo real
• 🗺️ Visualização do mapa interativo
• 🔎 Busca de destinos por endereço
• 🧭 Cálculo de rota até o destino
• ⏰ Disparo de alarme ao se aproximar do destino
• 🔔 Alarme sonoro automático ao atingir a distância configurada
• 📲 Instalação como aplicativo (PWA)
• 🌐 Funcionamento diretamente no navegador
• 📡 Atualização da posição durante o trajeto

📲 Instalação como Aplicativo (PWA)

O aplicativo pode ser instalado diretamente no celular:

Abra o site no navegador

Aguarde o botão Instalar Aplicativo

Toque em Instalar

O app será adicionado à tela inicial

Após instalado, o aplicativo funciona de forma semelhante a um app nativo.
